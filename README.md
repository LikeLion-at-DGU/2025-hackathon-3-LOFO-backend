## 📌 About LOFO

노포는 **홍보 역량 부족**으로, 청년은 **기회 부족**으로 어려움을 겪고 있습니다.  
**LOFO**는 두 문제를 연결해, **AI 기반 미션과 피드백**을 통해

-   청년에게는 **실전 경험과 성장 기회**를
-   상인에게는 **지속 가능한 홍보 자산**을 제공합니다.
